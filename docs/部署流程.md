# Web Plotter 工具部署流程

## 部署

这部分是给控制人员，进行小电脑并进行该工具的部署流程说明的，比较简单就简略说明

github URL：https://github.com/Aoalas/web_plotter

进入后clone下来，将web_plotter_backend这一功能包放入机器人工作空间内**（注意是web_plotter_backend不是web_plotter，即是将clone下来的包里面的一个文件夹放入工作空间，而不是直接将clone的仓库放入工作空间）**

接着按照README的提示，clone一个名为rospy_message_converter的功能包，放入工作空间内

同时为小电脑使用easywifi配置网络，下载其他前置内容

确定有网后，在终端进行apt的update，然后输入下面的指令

sudo apt install python3-flask python3-flask-socketio python3-eventlet

确认成功安装之后，回到工作空间

确定工作空间内含有前面说的两个功能包之后，输入catkin build编译，通过即可，一般不会报错



## 测试

完成source后，开一个终端，在终端输入

**rosrun web_plotter_backend server.py**

没问题的话后面基本只要rosrun一下就能用了，记得跟使用者说明白要读什么话题



## 简述

这个项目就是将ros的topic信息先打包成JSON数据，然后通过处理，在前端页面显示

server.py就是用来处理数据的程序，index.html就是用来显示页面的前端文件
